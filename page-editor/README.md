# PageCraft 에디터

드래그앤드롭으로 쇼핑몰 상세페이지를 만드는 에디터입니다.

---

## 🚀 Vercel 배포 방법 (코딩 불필요!)

### 1단계 — Vercel 가입
1. [vercel.com](https://vercel.com) 접속
2. **Sign Up** 클릭
3. **Continue with GitHub** 선택 (GitHub 계정 없으면 이메일로 가입)

### 2단계 — 프로젝트 업로드
1. 로그인 후 대시보드에서 **Add New → Project** 클릭
2. 화면 아래쪽 **"Or deploy from your local project"** 찾기
3. 이 ZIP 파일을 압축 해제한 폴더를 드래그앤드롭

### 3단계 — 배포 설정
- Framework Preset: **Vite** 선택
- Build Command: `npm run build` (자동 입력됨)
- Output Directory: `dist` (자동 입력됨)
- **Deploy** 버튼 클릭!

### 4단계 — 완료 🎉
- 약 1~2분 후 `https://page-editor-xxx.vercel.app` 주소 생성
- 이 주소를 누구에게나 공유 가능!

---

## 💻 로컬에서 실행하는 방법 (선택사항)

Node.js가 설치되어 있다면:

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속
